Подробное описание

Класс, который сканирует таблицу, на наличие листов для каждого сервера.

Аргументы
creatorSheetsОтвечает за создание google листа, в существующей google таблице.
writerSheetsОтвечает за запись данных в google лист.
workFilesОтвечает за запись данных в txt файл, если не удалось соединиться с google Api и сработало исключение.

Сканирует листы в существующей таблице.

Eсли лист не найден, вызывается метод объекта: ICreatorSheets

Eсли лист найден, вызывается метод объекта: IWriterSheets и данные у листа обновляются.

Аргументы
ListIdПередает Id google таблицы.
serversПередает список серверов с данными.
sheetServiceПередает сервис для взаимодействия с таблицами google.

Замещает IScanerSheets.
